# from\_angle

>  Creates an affine transformation matrix from the given 2D rotation `angle` (in
>  radians).
>  The resulting matrix can be used to transform 2D points and vectors. See
>  [`Self::transform_point2()`] and [`Self::transform_vector2()`].
