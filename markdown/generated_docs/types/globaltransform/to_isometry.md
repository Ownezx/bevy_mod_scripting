# to\_isometry

>  Returns the isometric part of the transformation as an [isometry]. Any scaling done by the
>  transformation will be ignored.
>  The transform is expected to be non-degenerate and without shearing, or the output
>  will be invalid.
>  [isometry]: Isometry3d
