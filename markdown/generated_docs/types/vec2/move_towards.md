# move\_towards

>  Moves towards `rhs` based on the value `d`.
>  When `d` is `0.0`, the result will be equal to `self`. When `d` is equal to
>  `self.distance(rhs)`, the result will be equal to `rhs`. Will not go past `rhs`.
