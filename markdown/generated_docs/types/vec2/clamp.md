# clamp

>  Component-wise clamping of values, similar to [`f32::clamp`].
>  Each element in `min` must be less-or-equal to the corresponding element in `max`.
>  # Panics
>  Will panic if `min` is greater than `max` when `glam_assert` is enabled.
