# is\_paused

>  Returns `true` if the stopwatch is paused.
>  # Examples
>  ```
>  # use bevy_time::*;
>  let mut stopwatch = Stopwatch::new();
>  assert!(!stopwatch.is_paused());
>  stopwatch.pause();
>  assert!(stopwatch.is_paused());
>  stopwatch.unpause();
>  assert!(!stopwatch.is_paused());
>  ```
