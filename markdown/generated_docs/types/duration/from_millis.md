# from\_millis

>  Creates a new `Duration` from the specified number of milliseconds.
>  # Examples
>  ```
>  use std::time::Duration;
>  let duration = Duration::from_millis(2_569);
>  assert_eq!(2, duration.as_secs());
>  assert_eq!(569_000_000, duration.subsec_nanos());
>  ```
