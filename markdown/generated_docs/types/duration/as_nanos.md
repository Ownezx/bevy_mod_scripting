# as\_nanos

>  Returns the total number of nanoseconds contained by this `Duration`.
>  # Examples
>  ```
>  use std::time::Duration;
>  let duration = Duration::new(5, 730_023_852);
>  assert_eq!(duration.as_nanos(), 5_730_023_852);
>  ```
