# as\_micros

>  Returns the total number of whole microseconds contained by this `Duration`.
>  # Examples
>  ```
>  use std::time::Duration;
>  let duration = Duration::new(5, 730_023_852);
>  assert_eq!(duration.as_micros(), 5_730_023);
>  ```
