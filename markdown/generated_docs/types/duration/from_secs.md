# from\_secs

>  Creates a new `Duration` from the specified number of whole seconds.
>  # Examples
>  ```
>  use std::time::Duration;
>  let duration = Duration::from_secs(5);
>  assert_eq!(5, duration.as_secs());
>  assert_eq!(0, duration.subsec_nanos());
>  ```
