# RotationGesture

### RotationGesture

1. f32

## Description

>  Two-finger rotation gesture.
> 
>  Positive delta values indicate rotation counterclockwise and
>  negative delta values indicate rotation clockwise.
> 
>  ## Platform-specific
> 
>  - Only available on **`macOS`** and **`iOS`**.
>  - On **`iOS`**, must be enabled first
