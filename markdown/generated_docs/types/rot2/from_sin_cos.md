# from\_sin\_cos

>  Creates a [`Rot2`] from the sine and cosine of an angle in radians.
>  The rotation is only valid if `sin * sin + cos * cos == 1.0`.
>  # Panics
>  Panics if `sin * sin + cos * cos != 1.0` when the `glam_assert` feature is enabled.
