# Capsule2d

### Capsule2d

- **radius** : f32
- **half\_length** : f32

## Description

>  A 2D capsule primitive, also known as a stadium or pill shape.
> 
>  A two-dimensional capsule is defined as a neighborhood of points at a distance (radius) from a line
